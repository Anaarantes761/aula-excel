# aula-excel
Este repositório é para praticar as habilidades do excel
Para uma empresa de roupas, é necessário os seguintes tipos de controle; funcionários, materiais e produtos.Tal empresa, é uma organização que busca reunuir pessoas e recursos para produzir/vender, com a intenção de lucrar. Para os funcionários, uma planilha para controle de horários de entrada, saída e faltas. Todas as funções definidas, sendo sempre acompanhadas para ver se estão sendo cumpridas, e reuniões para analisar os desempenhos semanalmente. Para os materiais, anotar a quantidade de tecidos, botões, e todos os materiais necessários que entram e saem da empresa, juntamente com o valor dos mesmos, e analisar o consumo para evitar desperdícios, tendo a certeza que o estoque seja suficiente para a confecção. Já para os produtos, fazer o controle do estoque de quantas peças entram e saem, os modelos mais vendidos e averiguar sempre a qualidade das roupas finalizadas antes das vendas.
